# deno-starter-kit

## getting started

- Install [Docker](https://www.docker.com/get-started).
- ```sh
  docker-compose up --build --detach
  docker attach deno-starter-kit_container
  ```
- ```sh
  make [ run | watch | debug | test | coverage | format | lint | doc | bundle ]
  ```
